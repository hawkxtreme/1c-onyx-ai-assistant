# Инструкция по работе с чатами (Onyx)

## Запуск

1. Откройте 1С:Предприятие
2. Перейдите **"Onyx" → "Чаты"**

## Стандартное использование

**Основное действие:**
1. Введите сообщение в поле **"Сообщение"**
2. Нажмите **"Отправить сообщение"** или клавишу **Enter**
3. Дождитесь ответа от ассистента

## Выбор агента

- Если агент не выбран, нажмите **"Выбрать агента"** и выберите нужного
- Если в системе один агент, он выбирается автоматически

## Создание нового чата

- Начните вводить сообщение и отправьте его - новая сессия создастся автоматически
- Или нажмите **"Новый чат"** / **"Создать сессию"**

## Выбор существующего чата

- Нажмите на нужный чат в списке слева
- История сообщений отобразится в центре формы


## Что происходит при отправке

1. Ваше сообщение появляется справа (синий цвет)
2. Появляется индикатор печати (ассистент формирует ответ)
3. Ответ ассистента появляется слева (белый цвет)

**Время ответа:** обычно 30-60 секунд

## Важно

- Перед отправкой сообщения должен быть выбран агент
- Сообщения сохраняются автоматически
- Ответы поддерживают форматирование (таблицы, списки, код)
